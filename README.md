# ReScript-React + TailWind Project Template

## Installation

```sh
npm install
```

## Run Project

```sh
yarn build:watch
# in another tab
yarn start
```
and visit http://localhost:3000/index.html to see your app in action.

## Build for Production

```sh
yarn release
```

## Test release

```sh
npx http-server dist
```
and visit http://localhost:8080.
